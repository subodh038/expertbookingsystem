# Real-Time Expert Session Booking System

A full-stack real-time booking platform where users can browse experts, view available time slots, and book sessions. The system prevents double booking and updates slot availability in real time using Socket.io.

---

## 🚀 Tech Stack

### Frontend
- React
- CSS (custom styling)
- Socket.io Client

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Socket.io

---

## ✨ Features

### User Features
- View list of experts with category, experience, and rating
- View expert details and available time slots grouped by date
- Book a session with form validation
- Prevent double booking for the same expert, date, and time slot
- Real-time slot updates across multiple users
- View personal bookings using email
- Booking status tracking (Pending / Confirmed / Completed)

### Technical Highlights
- RESTful API architecture
- Real-time communication using Socket.io
- MongoDB-based concurrency control
- Clean folder structure (routes / controllers / models)
- Environment variable support
